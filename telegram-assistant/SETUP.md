# 🤖 نظام الرد التلقائي على تيليجرام (بموافقتك) — دليل التركيب

نظام مجاني ١٠٠٪: **Google Apps Script + Telegram Bot + Gemini AI**.
العميل يبعت للبوت → الـ AI يكتب رد مقترح → يوصلك إنت بزرّين **✅ موافقة / ✏️ تعديل** → ما يتبعتش للعميل غير لما توافق.

التركيب ~١٥ دقيقة، كله من المتصفح، من غير برمجة.

---

## الخطوة ١: اعمل البوت (٢ دقيقة)
1. افتح تيليجرام وكلّم **@BotFather**.
2. ابعت `/newbot` → اختار اسم + يوزر (لازم ينتهي بـ `bot`).
3. هيديك **TOKEN** (سلسلة طويلة) — احفظها.

## الخطوة ٢: (مفيش مفتاح AI خالص ✅)
النظام بقى **بنك إجابات بكلمات مفتاحية** — مفيش AI ولا quota ولا أي مفاتيح. اتخطى الخطوة دي.

## الخطوة ٣: هات الـ Chat ID بتاعك (١ دقيقة)
1. كلّم البوت **@userinfobot** في تيليجرام.
2. هيديك رقم (Id) — ده **ADMIN_CHAT_ID** بتاعك.
3. **مهم:** اضغط Start على البوت بتاعك (اللي عملته في خطوة ١) عشان يقدر يبعتلك.

## الخطوة ٤: اعمل سكريبت جوجل (٥ دقيقة)
1. روح **script.google.com** → **New project**.
2. امسح أي كود موجود، والصق محتوى ملف **Code.gs**.
3. من **Project Settings (⚙️)** → نزّل لـ **Script Properties** → **Add script property** وضيف ٣ خصائص:
   | Property | Value |
   |----------|-------|
   | `TELEGRAM_TOKEN` | التوكن من BotFather |
   | `ADMIN_CHAT_ID` | رقمك من @userinfobot |
4. احفظ.

## الخطوة ٥: انشر كـ Web App (٢ دقيقة)
1. فوق على اليمين **Deploy → New deployment**.
2. النوع: **Web app**.
3. **Execute as:** Me · **Who has access:** **Anyone**.
4. **Deploy** → اقبل الصلاحيات (Authorize).
5. هيطلعلك **Web app URL** (مش محتاج تنسخه).

## الخطوة ٦: وصّل البوت (الطريقة المضمونة) ⚠️
> مهم: متستخدمش `setWebhook` من المحرر — بيحط لينك `/dev` اللي تيليجرام بيرفضه (401). استخدم لينك `/exec` يدويًا:

1. **Deploy → Manage deployments** → انسخ **Web app URL** (لازم ينتهي بـ **`/exec`**).
2. اتأكد إن **Who has access = Anyone** (لو لأ: Edit ✏️ → غيّرها → Deploy).
3. افتح ده في المتصفح (بدّل التوكن واللينك):
```
https://api.telegram.org/bot<التوكن>/setWebhook?url=<لينك_exec>
```
4. لازم يرجع `{"ok":true,...,"Webhook was set"}` ✅
5. تأكيد: افتح `https://api.telegram.org/bot<التوكن>/getWebhookInfo` → لازم الـ url ينتهي بـ `/exec` ومفيش `last_error_message`.

## جرّبه ✅
- من حساب تاني (أو صاحبك) كلّم البوت واسأل سؤال زي «الباقات إيه؟».
- المفروض **يوصلك إنت** رسالة فيها الرد المقترح + زرّين.
- اضغط **✅ موافقة** → الرد يروح للعميل. أو **✏️ تعديل** → اكتب ردك ويتبعت.

## ملاحظات
- **مجاني تمامًا** ضمن حدود Gemini المجانية (كافية لمئات الرسائل يوميًا).
- لو موضوع حساس (استرجاع/شكوى/طلب خاص) البوت بيعلّمه 🚩 عشان تراجعه إنت.
- لتعديل المعلومات/الأسعار اللي البوت بيرد بيها: عدّل نص `KNOWLEDGE` في أول الكود.
- لو غيّرت الكود: اعمل **Deploy → Manage deployments → Edit → New version**.
