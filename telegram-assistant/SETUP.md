# 🤖 نظام الرد على تيليجرام (بموافقتك) — دليل التركيب

نظام مجاني ١٠٠٪: **بنك إجابات بكلمات مفتاحية** (بدون AI). العميل يبعت → النظام يلقط الكلمة → يبعتلك الرد للمراجعة بزرّين [✅ موافقة] [✏️ تعديل] → ما يتبعتش للعميل غير بموافقتك.

> ⭐ **مهم:** استخدم نسخة **Cloudflare Workers** (`worker.js`) — دي الموثوقة. نسخة Google Apps Script (`Code.gs`) فيها مشكلة إن تيليجرام بيقف بعد أول رسالة (بسبب رد 302)، فمتنصحش بيها.

---

# 🟢 الطريقة الموصى بيها: Cloudflare Workers (worker.js)

## الخطوة ١: اعمل البوت
- كلّم **@BotFather** → `/newbot` → خد الـ **TOKEN**.
- خد رقمك (Chat ID) من **@userinfobot**. واضغط **Start** على البوت بتاعك.

## الخطوة ٢: اعمل Worker على Cloudflare (مجاني)
1. سجّل في **dash.cloudflare.com** (مجاني، بدون كارت).
2. **Workers & Pages → Create → Workers → Create Worker** → سمّيه (مثلاً empire-bot) → **Deploy**.
3. اضغط **Edit code** → امسح الكود الافتراضي → الصق محتوى **worker.js**.
4. فوق الكود، حط التوكن ورقمك في أول السطور:
   `const TELEGRAM_TOKEN = "..."` و `const ADMIN_CHAT_ID = "..."`.
5. **Deploy**. وانسخ رابط الـ Worker (شكله: `https://empire-bot.xxxx.workers.dev`).

## الخطوة ٣: وصّل البوت
افتح ده في المتصفح (بدّل التوكن ورابط الـ worker):
```
https://api.telegram.org/bot<التوكن>/setWebhook?url=<رابط_الworker>&drop_pending_updates=true
```
لازم يرجّع `{"ok":true,...}` ✅

## جرّب
- كلّم البوت **من حساب تاني** واسأل «الباقات إيه؟».
- المفروض **يوصلك إنت** الرد المقترح + زرّين → اضغط ✅ يروح للعميل.
- ابعت رسالة تانية وتالتة — هيفضل شغّال من غير ما يقف 🔥 (لأن Cloudflare بيرجّع 200).

➕ لإضافة سؤال جديد: في `worker.js` ضيف عنصر في `ANSWERS`: `{ keys:['كلمة'], reply:\`الرد\` }`.

# 🟡 (بديل قديم) Google Apps Script — غير منصوح بيه

---

## الخطوة ١: اعمل البوت (٢ دقيقة)
1. افتح تيليجرام وكلّم **@BotFather**.
2. ابعت `/newbot` → اختار اسم + يوزر (لازم ينتهي بـ `bot`).
3. هيديك **TOKEN** (سلسلة طويلة) — احفظها.

## الخطوة ٢: (مفيش مفتاح AI خالص ✅)
النظام بقى **بنك إجابات بكلمات مفتاحية** — مفيش AI ولا quota ولا أي مفاتيح. اتخطى الخطوة دي.

## الخطوة ٣: هات الـ Chat ID بتاعك (١ دقيقة)
1. كلّم البوت **@userinfobot** في تيليجرام.
2. هيديك رقم (Id) — ده **ADMIN_CHAT_ID** بتاعك.
3. **مهم:** اضغط Start على البوت بتاعك (اللي عملته في خطوة ١) عشان يقدر يبعتلك.

## الخطوة ٤: اعمل سكريبت جوجل (٥ دقيقة)
1. روح **script.google.com** → **New project**.
2. امسح أي كود موجود، والصق محتوى ملف **Code.gs**.
3. من **Project Settings (⚙️)** → نزّل لـ **Script Properties** → **Add script property** وضيف ٣ خصائص:
   | Property | Value |
   |----------|-------|
   | `TELEGRAM_TOKEN` | التوكن من BotFather |
   | `ADMIN_CHAT_ID` | رقمك من @userinfobot |
4. احفظ.

## الخطوة ٥: انشر كـ Web App (٢ دقيقة)
1. فوق على اليمين **Deploy → New deployment**.
2. النوع: **Web app**.
3. **Execute as:** Me · **Who has access:** **Anyone**.
4. **Deploy** → اقبل الصلاحيات (Authorize).
5. هيطلعلك **Web app URL** (مش محتاج تنسخه).

## الخطوة ٦: وصّل البوت (الطريقة المضمونة) ⚠️
> مهم: متستخدمش `setWebhook` من المحرر — بيحط لينك `/dev` اللي تيليجرام بيرفضه (401). استخدم لينك `/exec` يدويًا:

1. **Deploy → Manage deployments** → انسخ **Web app URL** (لازم ينتهي بـ **`/exec`**).
2. اتأكد إن **Who has access = Anyone** (لو لأ: Edit ✏️ → غيّرها → Deploy).
3. افتح ده في المتصفح (بدّل التوكن واللينك):
```
https://api.telegram.org/bot<التوكن>/setWebhook?url=<لينك_exec>
```
4. لازم يرجع `{"ok":true,...,"Webhook was set"}` ✅
5. تأكيد: افتح `https://api.telegram.org/bot<التوكن>/getWebhookInfo` → لازم الـ url ينتهي بـ `/exec` ومفيش `last_error_message`.
