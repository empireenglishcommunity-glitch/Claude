## 👋 أهلًا بيك في Empire English

ده مش جروب عادي. ده **نظام تعلّم يومي** هيخليك تتكلم إنجليزي بلهجة أمريكية صح.

**إيه اللي هتعمله كل يوم:**
- 7 مهام يومية (45 دقيقة بس)
- تمارين نطق + كلمات + استماع + كتابة + كلام
- AI بيديك feedback فوري
- نقاط + streaks + leaderboard

| الخطوة | إيه تعمل |

## 📋 الأوامر الأساسية (Commands)

اكتب الأوامر دي في قناة `#bot-commands`:

| الأمر | الوظيفة |
|-------|---------|
| `!join <هدفك>` | سجل نفسك وحدد هدفك |
| `!done <task>` | سجّل إنك خلصت مهمة (accent, vocab, shadow, speaking, listening, writing, community) |
| `!progress` | شوف تقدمك ونقاطك |
| `!streak` | شوف الـ streak بتاعك |
| `!level` | معلومات عن مستواك |
| `!week` | منهج الأسبوع الحالي |
| `!top` | لوحة المتصدرين (النقاط) |
| `!streaks` | لوحة المتصدرين (الاستمرارية) |
| `!help` | كل الأوامر |

## ⚠️ قوانين مهمة

1. **إنجليزي بس** — كل القنوات إنجليزي. استثناء: `#l0-questions` (سؤال واحد بالعربي)
2. **احترم الكل** — متتريقش على حد. الكل بدأ من الصفر
3. **الالتزام** — النظام بيشتغل لما انت تشتغل. 45 دقيقة يوميًا بس
4. **Voice Lounges** — لازم مايك. أقل حاجة 10 دقايق. متكلمش أكتر من دقيقتين متواصل
5. **متقفزش مستويات** — الترقية بالامتحان بس. مفيش استثناءات
